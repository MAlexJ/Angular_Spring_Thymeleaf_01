RESTful CRUD With AngularJS

This post will show how to perform typical CRUD (create, read, update and delete) operations in AngularJS when consuming a RESTful web service.

  • Example 1

    GET /phone/{id} (View) – получает полную информацию о объекте.

    id: {{phone.id}}

    title: {{phone.title}}

    description: {{phone.description}}

    price: {{phone.price | currency }}

    Example 2
    PUT /phone/ (Create) – создает новый объект.
    Example 3
    POST /phone/{id} (Edit) – изменяет данные с идентификатором {id}
    Example 4

    DELETE /phone/{id} (Delete) – удаляет данные с идентификатором {id}.

    Example 5

    GET /phone/ (Index) – получает список всех объектов.

    id: {{itemPhone.id}}

    title:{{itemPhone.title}}

    price: {{itemPhone.price | currency }}
